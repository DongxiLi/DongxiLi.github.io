再谈字符串

无标签

再谈字符串

字符串，即把字符按照一定的顺序串起来组成一个有限的序列。

Python中，我们使用''或者""包围起来，就可以表示一个字符串。字符串中的字符可以使特殊字符，英文字符，中文字符，希腊字母，Emoji字符等

character_name = 'DongxiLi'
character_name2 = '李东西'

# 使用三引号可以对字符串进行折行，而折行的样式就是输出的样式
character_name = '''
Dong
xi
Li
'''

转义字符和原始字符串

可以在字符串中使用\（反斜杠）来表示转义，也就是说\后面的字符不再是它原来的意义，例如：\n不是代表反斜杠和字符n，而是表示换行；\t也不是代表反斜杠和字符t，而是表示制表符。所以如果字符串本身又包含了'、"、\这些特殊的字符，必须要通过\进行转义处理。例如要输出一个带单引号或反斜杠的字符串，需要用如下所示的方法。

s = '\'hello, world'
print(s)

s2 = '\\hello, world\\'
print(s2)

Python字符串中可以用r或者R开头，这种字符串被称之为原始字符串，即字符串中的每一个字符都是它本来的含义，不存在转义。例如，在字符串'hello\n'中，\n表示换行；而在r'hello\n'中，\n不再表示换行，就是反斜杠和字符n。大家可以运行下面的代码，看看会输出什么。

# 字符串s中的\t是制表符,\n是换行符
s = '\time up \now'
print(s)

# 字符串s2中没有转义字符,每个字符都是原始含义
s2 = r'\time up \now'
print(s2)

字符串的运算

字符串的拼接和重复

# 字符串的拼接和重复
s = '李' + '东西'
print(s)  # 李东西

s2 = '呵呵' * 3
print(s2)  # 呵呵呵呵呵呵

s += s2  # s = s + s2
print(s)  # 李东西呵呵呵呵呵呵

比较运算符

字符串在计算机内存中也是以二进制形式存在的，那么字符串的大小比较比的是每个字符对应的编码的大小。例如A的编码是65， 而a的编码是97，所以'A' < 'a'的结果相当于就是65 < 97的结果，很显然是True；而'boy' < 'bad'，因为第一个字符都是'b'比不出大小，所以实际比较的是第二个字符的大小，显然'o' < 'a'的结果是False，所以'boy' < 'bad'的结果也是False。如果不清楚两个字符对应的编码到底是多少，可以使用ord函数来获得，例如ord('A')的值是65

s1 = 'a whole new world'
s2 = 'hello world'
print(s1 == s2, s1 < s2)      # False True
print(s2 == 'hello world')    # True
print(s2 == 'Hello world')    # False
print(s2 != 'Hello world')    # True

注意：

字符串比较运算符比较的事字符串的内容

身份运算符

is运算符(身份运算符)， 如果用is来比较两个字符串，它比价的是两个变量对应的字符串是否在内存中相同的位置(内存地址)。

s1 = 'hello world'
s2 = 'hello world'
s3 = s2

# 比较字符串的内容
print(s1 == s2, s2 == s3)  # True True

# 比较字符串的内存地址
print(s1 is s2, s2 is s3)  # False True

print(id(s1), id(s2), id(s3))  # 140422212438512 140422213227888 140422213227888

提示：

Python中可以用id来查看变量的内存地址

成员运算符

Python中可以用in和not in判断一个字符串是否存在内外一个字符或者字符串，in和not in运损通常称之为成员运算，会产生布尔值True或者False

s1 = 'helo world'
print('wo' in s1)  # True
s2 = 'googbye'
print(s2 in s1)  # False

获取字符串的长度

name = 'DongxiLi'
print(len(name))  # 8

索引和切片

通过索引可以从字符串中取出某个字符，运算符是[n] 这里的0 <= n < N，这里的N指的是字符串的长度；当然这里的n也可以-N <= n <0，当n负数是通常称之为负向索引

字符串是不可变类型， 所以不能通过索引修改字符串中的字符

s = 'abc123456'
N =len(s)

# 获取第一个字符串
print(s[0], s[-N])  # a a

# 获取最后一个字符
print(s[N-1], s[-1])  # 6 6

# 获取字符c
print(s[2], s[-7])  # c c

# 获取字符3
print(s[5], s[-4])

注意：

进行索引操作时一定要注意索引的范围，不然会发生索引越界，引发IndexError异常，错误提示信息为：string index out of range（字符串索引超出范围

如果要字符串中取出多个字符，则可以通过切片，运算符是[i:j:k]其中i是开始索引，j是结束索引(取不到j),k是步长，默认步长为1，表示从前向后获取相邻字符的连续切片，所以:k部分可以省略。 

s = 'abc123456'

# i=2, j=5, k=1 的正向切片
print(s[2:5])  # c12

# i=-7, j=-4, k=1 的正向切片
print(s[-7:-4])  # c12

print(s[2:])  # c123456

# i=-7, j=9, k=1的正向切片操作
print(s[-7:])  # c123456

# i=2, j=9, k=2的正向切片操作
print(s[2::2])  # c246

# i=-7, j=9, k=2的正向切片操作
print(s[-7::2])  # c246

# i=7, j=1, k=-1的负向切片操作
print(s[7:1:-1])  # 54321c

# i=-2, j=-8, k=-1的负向切片操作
print(s[-2:-8:-1])  # 54321c

# i=7, j=-10, k=-1的负向切片操作
print(s[7::-1])  # 54321cba

# i=-1, j=1, k=-1的负向切片操作
print(s[:1:-1])  # 654321c

# i=-1, j=-10, k=-1的负向切片
print(s[::-1])      # 654321cba

# i=-1, j=-10, k=-2的负向切片
print(s[::-2])      # 642ca

字符串的方法

Python中，我们可以通过字符串类型自带的方法对字符串进行操作和处理，对于一个字符串类型的变量，我们可以用变量名.方法名()的方式来调用它的方法。

大小写相关操作

name = 'dong xi li'

# 使用capitalize方法获得字符串首字母大写后的字符串
print(name.capitalize())  # Dong xi li

# 使用title方法获得字符串每个单词首字母大写后的字符串
print(name.title())  #Dong Xi Li

# 使用upper方法获得字符串大写后的字符串
print(name.upper())  # DONG XI LI

name2 = ’DONG XI LI‘

# 使用lower方法获得字符串小写后的字符串
print(name.lower)  # 'dong xi li'

查找操作

如果想在一个字符串中从前往后查找有没有另外一个字符串，可以使用find或者index方法

s = 'hello, world!'

# find方法从字符串中查找另外一个字符串的所在位置
# 找到了就放回字符串中另一个字符串字符的索引
print(s.find('or'))  # 8

# 找不到返回-1
print(s.find('DongxiLi'))  # -1
# index方法与find方法类似
# 找到了返回字符串中另一个字符串首字符的索引
print(s,inex('or'))
# 找不到引发异常
print(s.index('DongxiLi'))  # ValueError:substring not found

在使用find和index方法时还可以通过方法的参数来指定查找的范围，即可以指定开始的索引。find和index方法还有逆向查找的版本。分别是rdind和index。

s = 'hello world'

# 从前向后查找o出现的位置(相当于第一出现)
print(s.find('o')) # 4

# 从索引为5的位置开始查找o出现的位置
print(s.find('0', 5))  # 7

# 从后向前查找字符o出现的位置(相当于最后一次出现)
print(s.rfind('o'))  # 7

可以通过字符串的startswith，endswith来判断字符串是否以某个字符开头或者结尾；也可以用is开头的方法来判断字符串的特征。

s1 = 'hello, world'

# startwith方法检查字符串是否以指定字符串开头，返回bool
print(s1.startswith('H'))  # False
print(s1.startswith('he'))  # True

# endswith方法检查字符串是否以指定字符串结尾，返回bool
print(s1.endswith('d'))  # True

s2 = 'abc123'

# isdigit方法检查字符串是否有数字构成，返回bool
print(isdigit())  # False
# isalpha方法检查字符串是否由字母构成，返回bool
print(s2.isalpha())  # False
# isalnum方法检查字符串是否以数字和字母构成，返回bool
print(s2.isalnum())  #True

格式化字符串

Python中字符串类型可以通过center,ljust,rjust方法做居中，左对齐和右对齐的处理。

s = 'hello, world'

# center方法以宽度20将字符串居中并在两侧填充*
print(s.center(20, '*'))  # ****hello world*****

# rjust方法以宽度为20将字符串右对齐并在左侧填充-  
print(s.rjust(20, '-'))  # ---------hello world

# ljust方法以宽度为20将字符串左对齐并在右侧填充^  
print(s.ljust(20, '^'))  # hello world^^^^^^^^^

修剪操作

字符串的strip方法可以帮助我们将原字符串修剪掉左右两端的空格之后的字符串。这个方法非常有实用价值，通常用来将用户输入中因为不小心键入的头尾空格去掉，strip方法还有lstrip和rstrip两个版本。

​```Python
s = '   jackfrued@126.com  \t\r\n'
# strip方法获得字符串修剪左右两侧空格之后的字符串
print(s.strip())    # jackfrued@126.com
​```
